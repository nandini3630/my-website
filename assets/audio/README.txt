Audio Files Directory

To add your song:
1. Place your audio file here and name it "song.mp3"
2. Supported formats: MP3, WAV, OGG
3. The "Our Song" button will automatically detect and play the file

Example:
- song.mp3 (your main song)
- song.wav (alternative format)

The audio player will work once you add a valid audio file to this directory.
